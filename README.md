## 📚 אודות המאגר

**LearningHub** הוא מאגר חינמי ופתוח של מדריכים וקורסים מקצועיים בעברית, המיועדים ללומדים בכל הרמות - ממתחילים ועד מתקדמים. המאגר כולל **38+ מדריכים מקיפים** ו-**12 קורסים מובנים** בנושאים מגוונים בעולם הטכנולוגיה והחדשנות.

### ✨ תכונות עיקריות

- 📖 **38+ מדריכים מקצועיים** - תוכן מעמיק ואיכותי
- 🎯 **12 קורסים מובנים** - למידה שיטתית עם תרגולים
- 🎤 **15 הקראות אודיו** - למידה נגישה ונוחה
- 🎨 **עיצוב מקצועי** - ממשק נקי וידידותי למשתמש
- 📱 **Responsive Design** - מותאם לכל המכשירים
- 🔍 **SEO מתקדם** - אופטימיזציה מלאה למנועי חיפוש
- ♿ **נגישות** - תמיכה בקוראי מסך ותקני WCAG

## 🚀 התחלה מהירה

### דרישות מקדימות

- דפדפן מודרני (Chrome, Firefox, Safari, Edge)
- חיבור אינטרנט
- אין צורך בהתקנה!

### שימוש במאגר

1. **גלישה לפי קטגוריות**
   ```
   פתח את categories.html לצפייה בכל המדריכים לפי נושאים
   ```

2. **חיפוש מדריך**
   ```
   השתמש ב-sitemap.html למפה מלאה של כל התכנים
   ```

3. **התחלת קורס**
   ```
   נווט ל-courses/ ובחר את הקורס המתאים
   ```

### פיתוח מקומי

אם ברצונך להריץ את המאגר מקומית:

```bash
# שיבוט המאגר
git clone https://github.com/bdnhost/-Resources-guide.git

# כניסה לתיקייה
cd -Resources-guide

# התקנת תלויות (אופציונלי)
npm install

# יצירת manifest אוטומטי
npm run generate

# פתיחת האתר
# פשוט פתח את categories.html בדפדפן
```

## 📖 הנחיות תרומה

אנו מזמינים אותך לתרום למאגר! ראה את הקובץ [CONTRIBUTING.md](CONTRIBUTING.md) להנחיות מפורטות.

### איך לתרום?

1. **Fork** את המאגר
2. צור **branch** חדש למשימה שלך
   ```bash
   git checkout -b feature/amazing-guide
   ```
3. **Commit** את השינויים
   ```bash
   git commit -m "Add: מדריך חדש על נושא X"
   ```
4. **Push** ל-branch
   ```bash
   git push origin feature/amazing-guide
   ```
5. פתח **Pull Request**

### רעיונות לתרומה

- ✍️ כתיבת מדריכים חדשים
- 🐛 תיקון באגים וטעויות
