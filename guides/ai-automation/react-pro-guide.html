זה מדריך דוגמה שמחזיק בעצמו סיווג JSON בתחילתו

🎯 כיצד זה עובד?

  1. המדריך כולל JSON comment בתחילת הקובץ
  2. כאשר מריצים npm run auto, הסקריפט קורא את ה-JSON
  3. הוא משתמש בקטגוריה שהוגדרה במקום לנחש
  4. הקטלוג מעודכן עם הסיווג הנכון

💡 למשתמש:

בעת יצירת מדריך חדש:

  1. העתק את master_guide_template.html
  2. אם לו את ה-JSON comment בתחילת הקובץ עם הפרטים שלך
  3. כתוב את תוכן המדריך
  4. הרץ npm run auto - הכל מעודכן אוטומטית!
