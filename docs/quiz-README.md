# 🎯 תבנית מבחן "בחן את עצמך"

תבנית HTML מוכנה לשימוש ליצירת מבחנים אינטראקטיביים במדריכים, עם אינטגרציה ל-EduManage API.

---

## 📋 תכונות

- ✅ **שאלות רב-ברירה** - תמיכה בכמות לא מוגבלת של שאלות
- 🎨 **עיצוב מקצועי** - ממשק נקי וידידותי למשתמש
- 📊 **חישוב ציון אוטומטי** - משוב מיידי עם אחוזים
- ✓/✗ **סימון תשובות** - תשובות נכונות ושגויות מסומנות
- 🔗 **אינטגרציה עם EduManage** - שמירת תוצאות במערכת
- 👤 **פורטל תלמיד** - קישור ישיר לפורטל אחרי סיום
- 📱 **Responsive** - מותאם לכל המכשירים

---

## 🚀 איך להשתמש?

### שלב 1: העתקת התבנית

```bash
cp docs/quiz-template.html path/to/your-quiz.html
```

### שלב 2: עדכון התצורה

פתח את הקובץ ומצא את `QUIZ_CONFIG`. עדכן את הפרמטרים:

```javascript
const QUIZ_CONFIG = {
    guideName: 'מדריך ChatGPT',     // שם המדריך שלך
    guideChapter: 'פרק 1',            // מספר הפרק
    guideSection: 'יסודות',          // שם הסעיף
    questions: [
        {
            question: 'מהו ChatGPT?',
            options: [
                'מודל שפה מבוסס בינה מלאכותית',
                'דפדפן אינטרנט',
                'מערכת הפעלה',
                'שפת תכנות'
            ],
            correct: 0  // אינדקס התשובה הנכונה (0 = תשובה ראשונה)
        },
        // הוסף שאלות נוספות...
    ]
};
```

### שלב 3: הוספת השאלות שלך

כל שאלה צריכה להכיל:

```javascript
{
    question: 'טקסט השאלה',          // שאלה בעברית ברורה
    options: [                        // מערך של 2-5 אפשרויות
        'תשובה 1',
        'תשובה 2',
        'תשובה 3',
        'תשובה 4'
    ],
    correct: 2                        // אינדקס התשובה הנכונה (מתחיל מ-0)
}
```

## 🔧 התאמה אישית

### שינוי צבעים

ערוך את ה-CSS בקובץ:

```css
/* צבע ראשי */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* תשובה נכונה */
.option.correct {
    border-color: #10b981;
    background: #d1fae5;
}

/* תשובה שגויה */
.option.incorrect {
    border-color: #ef4444;
    background: #fee2e2;
}
```

### שינוי טקסטים

```javascript
// טקסט כפתור
<button type="submit">✅ בדוק תשובות</button>

// הודעות לפי ציון
if (score.percentage >= 90) {
    icon = '🏆';
    message = 'מעולה! אתה שולט בחומר!';
}
```

### הוספת טיימר (אופציונלי)

```javascript
let timeLeft = 600; // 10 דקות בשניות

const timer = setInterval(() => {
    timeLeft--;
    document.getElementById('timer').textContent =
        `${Math.floor(timeLeft / 60)}:${(timeLeft % 60).toString().padStart(2, '0')}`;

    if (timeLeft <= 0) {
        clearInterval(timer);
        document.getElementById('quiz-form').dispatchEvent(new Event('submit'));
    }
}, 1000);
```

## 🌐 אינטגרציה עם EduManage

### מה נשלח ל-API?

```json
{
    "student_id": "12345",
    "guide_name": "מדריך ChatGPT",
    "guide_chapter": "פרק 1",
    "guide_section": "יסודות",
    "guide_task": "מבחן בחן את עצמך",
    "page_url": "https://bdnhost.net/Resources/guides/ai-automation/quiz.html",
    "form_data": {
        "quiz_score": 80,
        "correct_answers": 4,
        "total_questions": 5,
        "answers": [0, 1, 2, 1, 2],
        "timestamp": "2025-12-24T10:30:00.000Z"
    }
}
```

### מה קורה אחרי שליחה?

1. **הצלחה** ✅
   - התוצאות נשמרות ב-EduManage
   - התלמיד מקבל קישור לפורטל האישי
   - ניתן לצפות בהיסטוריית המבחנים

2. **כישלון** ❌
   - הודעת שגיאה מוצגת
   - התוצאות נשמרות מקומית
   - אפשרות לנסות שוב

## 📊 דוחות ומעקב

### צפייה בתוצאות ב-EduManage

לאחר שליחה מוצלחת, ניתן לצפות ב:

- **ציון אחרוני** - הציון במבחן
- **היסטוריה** - כל הניסיונות הקודמים
- **ניתוח תשובות** - תשובות נכונות/שגויות
- **זמן השלמה** - כמה זמן לקח
- **התקדמות** - באילו מדריכים עברת מבחנים

## ⚠️ שגיאות נפוצות

### שגיאה: "נא להזין מזהה תלמיד"

**פתרון**: ודא שהתלמיד מזין מזהה בשדה המתאים לפני שליחה.

### שגיאה: "student_id לא קיים במערכת"

**פתרון**: המזהה חייב להיות קיים במערכת EduManage. צור תלמיד חדש או השתמש במזהה קיים.

### שגיאה: "שגיאת חיבור"

**פתרון**: בדוק חיבור לאינטרנט ו-CORS policy של ה-API.

### הכפתור לא עובד

**פתרון**: ודא שכל השאלות נענו (כל input[required] מסומן).

## 🎓 שימושים מומלצים

### מתי להשתמש במבחן?

✅ **כן - מומלץ:**
- בסיום כל פרק במדריך
- לבדיקת הבנה של מושגי יסוד
- לפני מעבר לחומר מתקדם
- כתרגול לפני מבחן אמיתי

❌ **לא - לא מומלץ:**
- כמבחן סופי יחיד
- עם יותר מ-15 שאלות (יהיה ארוך מדי)
- לחומר שטרם נלמד
- כתחליף ללמידה

### כמה שאלות להוסיף?

- **קצר** (5 שאלות) - לבדיקה מהירה
- **בינוני** (10 שאלות) - לבדיקה רגילה
- **ארוך** (15 שאלות) - לבדיקה מקיפה
