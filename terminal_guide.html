Terminal Guide

המדריך המלא לשורת הפקודה (CMD/PowerShell) למהנדסים: ניהול קבצים, רשתות וסקריפטים.

למה להשתמש בטרמינל?

למרות שנוח להשתמש בעכבר, מהנדסים משתמשים בשורת הפקודה כי היא: 1. מהירה יותר: ביצוע פעולות מורכבות בשניות. 2. ניתנת לאוטומציה: ניתן לכתוב סקריפטים שמבצעים משימות רוטיניות. 3. חיונית לפיתוח: הרצת Python, Git וחיבור לשרתים נעשים בעיקר מכאן.

בווינדוס, ניתן לפתוח את הטרמינל ע"י חיפוש "CMD" או "PowerShell" בתפריט התחלה.

💡 חומר למחשבה

האם ידעת ש... האקרים בסרטים תמיד משתמשים בטרמינל לא בגלל שזה נראה מגניב (אם כי זה כן), אלא כי זה באמת הכלי החזק והמהיר ביותר לשליטה במחשב!

ניווט וניהול קבצים

הפקודות החשובות ביותר שצריך להכיר (עובד ב-PowerShell ו-CMD):

פקודה הסבר דוגמה
ls או dir הצגת רשימת הקבצים בתיקייה הנוכחית ls
cd שינוי תיקייה (Change Directory) cd Project_Files
cd .. חזרה לתיקייה הקודמת (אחת אחורה) cd ..
mkdir יצירת תיקייה חדשה (Make Directory) mkdir New_Project
rm או del מחיקת קובץ (זהירות! אין סל מחזור) del old_file.txt

⚠️ רגע, עצור!

לפני שתמשיך... זכור שפקודת rm או del מוחקת קבצים לצמיתות! אין סל מחזור בטרמינל. תמיד בדוק פעמיים לפני מחיקה!

תרגול מהיר

# 1. צור תיקייה חדשה בשם 'Demo'
mkdir Demo

# 2. היכנס לתיקייה
cd Demo

# 3. צור קובץ טקסט חדש (PowerShell)
echo "Hello World" > hello.txt

# 4. הצג את תוכן התיקייה
ls

בדיקות רשת

פקודות לאבחון תקלות אינטרנט או חיבור לרכיבים תעשייתיים.

בדיקת כתובת IP (ipconfig)

ipconfig

פקודה זו תראה לך את כתובת ה-IP המקומית שלך (למשל 192.168.1.15).

בדיקת חיבור (Ping)

בדיקה האם יש תקשורת עם שרת או מכשיר אחר.

# בדיקת חיבור לגוגל
ping google.com

# בדיקת חיבור למכונה ברשת המקומית
ping 192.168.1.50

טיפים למתקדמים (PowerShell)

השלמה אוטומטית (Tab)

אל תקליד שמות ארוכים! התחל לכתוב את שם הקובץ ולחץ על Tab להשלמה אוטומטית.

היסטוריית פקודות

לחץ על החץ למעלה ↑ כדי לחזור לפקודות קודמות שהרצת.

ניקוי מסך

המסך עמוס מדי? כתוב cls או clear כדי לנקות אותו.

התקנת תוכנות (winget)

בווינדוס 10/11 ניתן להתקין תוכנות ישירות מהטרמינל!

# התקנת Chrome
winget install Google.Chrome

# התקנת VS Code
winget install Microsoft.VisualStudioCode

🤫 סוד קטן

ניסית פעם...? ללחוץ על החץ למעלה ↑ בטרמינל? זה מביא את הפקודה האחרונה שהרצת. אפשר לעבור על כל ההיסטוריה ולחסוך הרבה הקלדה!

🎯 מסקנות ויישום מעשי

✅ מה למדנו היום?

  • הטרמינל זה לא מפחיד - רק כלי עבודה חזק
  • 5 פקודות בסיסיות - ls, cd, mkdir, rm, ping פותרות רוב הצרכים
  • מהירות ויעילות - מה שלוקח דקות בעכבר, לוקח שניות בטרמינל
  • אוטומציה - אפשר לכתוב סקריפטים למשימות חוזרות
  • כלי מקצועי - חיוני לפיתוח, Git, ושרתים

🚀 איך להמשיך להתפתח?

  1. תרגול יומי - השתמש בטרמינל למשימה אחת ביום
  2. למד קיצורי מקלדת - Tab, ↑↓, Ctrl+C
  3. נסה PowerShell ISE - סביבה גרפית לסקריפטים
  4. למד Batch/PowerShell - כתיבת סקריפטים
  5. התקן Windows Terminal - טרמינל מודרני ויפה

⚠️ טעויות נפוצות להימנע מהן

  • מחיקה בלי בדיקה - תמיד וודא מה אתה מוחק
  • לא להשתמש ב-Tab - השלמה אוטומטית חוסכת זמן
  • פחד מלנסות - אי אפשר לשבור את המחשב
  • לא לזכור פקודות - כתוב רשימה או השתמש בהיסטוריה
  • להתעלם מהודעות שגיאה - תמיד קרא מה כתוב

📋 משימות לשבוע הקרוב

🎯 רמה בסיסית (מתחילים)

  • פתח CMD או PowerShell ונווט בין תיקיות עם cd
  • צור תיקייה חדשה עם mkdir ומחק אותה עם rmdir
  • השתמש ב-ls או dir לראות קבצים
  • נסה ping google.com לבדוק חיבור אינטרנט
  • השתמש ב-Tab להשלמה אוטומטית של שמות קבצים

⚡ רמה מתקדמת (מנוסים)

  • התקן Windows Terminal מה-Microsoft Store
  • למד פקודות מתקדמות: grep, find, tasklist
  • כתוב סקריפט Batch פשוט לגיבוי קבצים
  • נסה winget להתקנת תוכנות
  • למד Git - עבור למדריך Git
  • התחל עם Python - עבור למדריך Python

😎 בונוס

תרגיל קטן: נסה לכתוב tree בטרמינל (אם זה לא עובד, התקן עם winget install tree). זה מראה את מבנה התיקיות כמו עץ - מאוד יפה!
